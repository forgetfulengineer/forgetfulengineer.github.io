strtotime() - 將字串轉換為時間戳記

1. 將各種時間字串（如 yesterday、+1 week）轉換為 Unix 時間戳
2. 如果執行日期運算的環境沒有設定自己的時區或是想指定時區，可以使用 date_default_timezone_set() 調整。
3. 具體的日期時間取得時間戳不受 strtotime() 的第二個參數 $baseTimestamp 影響
4. 相對時間的時間字串會受 strtotime() 的第二個參數 $baseTimestamp 影響

常見日期時間字串應用：

絕對時間（Absolute Dates）
