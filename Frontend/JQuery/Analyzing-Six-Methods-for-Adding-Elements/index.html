【jQuery】解析六種添加元素方法

2025-05-09回覆文章分享

本篇重點

  • 常見的添加元素方法
  • 添加到指定元素內部：.append()、.prepend()、.appendTo()、.prependTo()
  • 添加到指定元素外部：.after()、.before()
  • 介紹各方法的使用效果及適用情境

開發網頁互動功能或動態內容時，動態新增 HTML 元素是常見的需求，例如：點擊按鈕時添加圖片、在特定區塊加入提示訊息等。

在 jQuery 中，為了讓這些操作更簡潔有效率，提供了多種用來「新增元素」的方法，如 .append()、.prepend()、.after()、.before()、.appendTo() 與 .prependTo()，雖然功能相似，但添加的位置與語法的寫法略有不同！
